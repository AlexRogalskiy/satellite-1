# Whereami

With this simple service the cloud provider can be determined easily with a simple HTTP call.

## Supported Cloud Providers

- Amazon
- Alibaba Cloud
- Azure
- Google Cloud
- Oracle Cloud
- DigitalOcean

## API

- GET "/whereami" -> {name: cloudprovider}
